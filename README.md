# SociActive API

A Node.js/Express REST API enabling users to link up for outdoor activities.

## Features
- Host, Join, and Manage Outdoor Activities (Hiking, Cycling, etc.).
- Filter upcoming activities.
- Review completed activities.
- **Dual Data Mode**: Runs with MongoDB (if configured) OR defaults to hardcoded Mock Data automatically.
